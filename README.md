# Plugin Basics

This repository contains resources and starter material for building Framer plugins.

## Getting Started

1. Open the project in your editor of choice.
2. Review `PLUGIN_TEMPLATE_GUIDE.md` for an overview of the architecture template.
3. Consult `Framer_Plugin_reference.md` and `Framer_component_reference.md` for API details.

## Publishing

After making your changes:

1. Stage and commit updates with Git.
2. Publish the repository using GitHub Desktop or the command line.
